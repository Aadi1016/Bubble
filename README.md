# Bubble Inc ThoughtSpot TML Repo

This repo contains TML files for Bubble Inc’s ThoughtSpot setup.

## Contents
- `AnalystStudio_connection.connection.tml`: TS connection to AnalystStudio DB
- `Bubble_Inc_Model.model.tml`: Logical data model
- `dim_client.table.tml`: Client dimension
- `dim_subscription.table.tml`: Subscription dimension
- `fact_sales.table.tml`: Sales fact table
- `Manifest.yaml`: To import the entire bundle easily into another ThoughtSpot instance
